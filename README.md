# File download

A file download Python3 script which checks file size,
HTTP status codes and takes HTTP ETag into account.


## Usage

``` bash
python file_download.py <url> <file_path>
```

## Output files

- the downloaded file as `<file_path>`
- the ETag as `<file_path>.etag`
